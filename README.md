# Go와 Gemini를 활용하여 League of Legends Assistant Discord Bot 만들기

## Install
```shell
go get -u github.com/bwmarrin/discordgo
go get -u github.com/google/generative-ai-go
go get -u github.com/joho/godotenv
```
